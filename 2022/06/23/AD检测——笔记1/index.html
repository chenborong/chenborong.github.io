    AD检测——笔记1

    1.介绍

    本次AD检测，采用常规的AD检测电路，使用单电阻分压，优点是电路和程序相对简单，缺点是对于个别的温度段分辨率不高，因此只适合对温度显示要求不高的场合。

    2.电路介绍

    如图所示，电路采用2.7K下拉，分压后使用104电容滤波储能，之所以使用2.7K，因为本次AD读取要求的从0~460F华氏度，范围较广，因此使用2.7K电阻

    3.真值表

    在写程序之前，还需要通过传感器的真值表，结合下来2.7K电阻，计算出各个温度的AD值，结合使用的单片机，选择8位AD或者12位AD，这里使用的是12位AD。

    以3950为例，在excel中，先列出每个温度点的电阻值，12位的AD计算公式为：1023/(R(temp)+2.7)*2.7 而8位的AD计算公式为：255/(R(temp)+2.7)*2.7

    得到没一个华氏度对应的AD之后，则可以开始准备写程序

    4.3AD转换，得到12位AD

    /****************************************
函数名称:uchar AD_Conbert(uchar ch_way)
输入:AD通道
输出:_sadoh _sadol(12位)
函数功能：AD转换,输入通道,返回AD值的高8位
*****************************************/
uint AD_Conbert(uchar ch_way)
{
    uint main_12 = 0;
    
    _sadc0 = ch_way;
    _start = 0;
    GCC_NOP();
    _start = 1;
    GCC_NOP();
    _start = 0;
    while (_adbz);
    main_12 = _sadoh;
    main_12 <<= 8;
    main_12 |= _sadol;
    main_12 >>= 4;
    return main_12;
}

    4.4削抖，滤波

    经过累计求平均的方法，去除最值，削抖，可以得到相对稳定的AD值

    /****************************************
函数名称:void Get_Temper()
输入:
输出:Temper
函数功能：将12位AD值，转化成华氏度(F)
*****************************************/
void Get_Temper()
{
    uchar i;
    //获取AD完毕
    if(b_Adc)
    {
        if(TEMP_MAIN2 >= 3707) {Temper = 460;b_Adc = 0;}	//短路,显示460F
        else if(TEMP_MAIN2 <= 20) {Temper = 20;b_Adc = 0;}	//开路,显示20F
        else 
        {
            for(i = 0;i <= 20;i++)
            {
                if(TEMP_MAIN2 <= Tab_temp_array[Loop_Arr])	
                {
                        Temper = Loop_Arr;			//Temper 为实际的温度
                        Loop_Arr = 460;
                        break;
                }
                if(++Loop_Arr >= 461)	//超过表格
                    break;
            }
            if(Loop_Arr >= 460) {b_Adc = 0;}
        }	
    }
}
